byte ButtonAndLED::getBtnState ( )

Function to get button state.

Returns
0 or BUTTON_IDLE constant if the button is not pressed. 1 or BTN_CLICK constant if the button is pressed once. 2 or BTN_DOUBLE_CLICK constant is the button is pressed twice. 3 or BTN_LONG_PRESS constant if the button is long pressed. 4 or BTN_LONG_RELEASE constant when the button is released after a long press.
